# UX TRACÉA — VERSION V1 (VERROUILLÉE)

## Positionnement
TRACÉA est un outil de régulation émotionnelle en temps réel.

Ce n’est :
- ni une thérapie
- ni un coaching
- ni une analyse

C’est une traversée guidée, centrée sur le corps.

## Architecture produit

### Double flow actif
- Flow court → forte activation / faible bande passante
- Flow long → activation modérée / disponibilité

### Routage
- "Très chargé" / "Assez chargé" → flow court
- "Un peu chargé" / "Plutôt calme" → flow long

## Flow long — doctrine
- pas de miroir IA intermédiaire
- IA uniquement en synthèse finale
- progression : Traverser → Reconnaître → Ancrer → Écouter → Émerger → Aligner

## Principes UX

### Simplicité radicale
- 1 écran = 1 action
- zéro surcharge

### Corps avant mental
- priorité au ressenti

### Micro-réception
- "Ok", "D'accord", "Noté"
- jamais bloquant

### Micro-rythme
- "Prends juste une seconde."
- aucun timer

### Continuité sensorielle
- "Et là, sous ce qui s’est un peu posé…"
- "À partir de ce qui s’est éclairci…"

### Émerger
- actions simples, naturelles

### Clôture
- "C’est suffisant pour maintenant."

## Synthèse finale
- descriptive uniquement
- aucun score
- fallback sobre

## Traces
- non analytique
- basé sur vécu

## Interdits
- pas de jargon
- pas d’analyse
